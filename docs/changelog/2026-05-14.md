## ⚠️ Supabase 마이그레이션 필요 (코드 배포 전 선행)

```sql
-- 1. 프로젝트 종료 상태 (#3)
ALTER TABLE sm_projects ADD COLUMN IF NOT EXISTS status TEXT DEFAULT 'active';
ALTER TABLE sm_projects ADD COLUMN IF NOT EXISTS completed_at TIMESTAMPTZ;

-- 2. Project Feed 노트 테이블 (#1)
-- ※ sm_tasks.id = BIGINT, sm_users.id = BIGINT, sm_projects.id = UUID
CREATE TABLE IF NOT EXISTS sm_notes (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  project_id    UUID NOT NULL REFERENCES sm_projects(id) ON DELETE CASCADE,
  task_id       BIGINT REFERENCES sm_tasks(id) ON DELETE SET NULL,
  type          TEXT NOT NULL CHECK (type IN ('progress','issue','meeting','handoff')),
  title         TEXT,
  content       TEXT,
  from_user_id  BIGINT REFERENCES sm_users(id) ON DELETE SET NULL,
  to_user_id    BIGINT REFERENCES sm_users(id) ON DELETE SET NULL,
  meeting_date  DATE,
  attendees     BIGINT[],
  created_at    TIMESTAMPTZ DEFAULT now()
);

ALTER TABLE sm_notes ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_all" ON sm_notes FOR ALL USING (true) WITH CHECK (true);
```

## 기능별 상세

### #4 Archive Box 버그 수정
**원인**: `archived` 탭 전환 시 `TaskListView` 사용 → `todo/in_progress/done` 그룹만 렌더링 → `archived` 상태 태스크가 어떤 그룹에도 속하지 않아 빈 화면 출력

**수정**: `archived` 탭을 `ArchiveView` 컴포넌트로 교체

---

### #2 아카이브 뷰 + 캘린더 잔류

**ArchiveView 기능**:
- 월별 그룹핑
- 프로젝트 필터 + 텍스트 검색
- 완료 시각 표시 (`timeAgo`)
- 복구 버튼 (→ `done` 상태로 복원)

**캘린더 archived 잔류**:
- `calendarTasks = allTasks` (archived 포함) → CalendarPanel에 전달
- `filteredTasks = activeTasks` (archived 제외) → Board/Tasks에 전달
- archived 태스크는 캘린더에서 **회색 + 취소선 + 불투명도 65%** 로 표시
- archived 태스크는 캘린더 드래그 불가 (dragItem: null)

---

### #3 프로젝트 종료 상태

**AdminPage**:
- 각 프로젝트에 **"종료"** 버튼 추가 → `status = 'completed'`, `completed_at = now()` 저장
- 종료된 프로젝트에 **"종료됨"** 뱃지 + **"재개"** 버튼 표시
- 종료 전 confirm 경고 ("캘린더에는 유지됩니다")

**AppSidebar**:
- 사이드바 Projects 섹션: `status !== 'completed'` 필터 → 활성 프로젝트만 표시
- 종료된 프로젝트가 1건 이상이면 `"종료된 프로젝트 N건"` 토글 표시
- 토글 펼치면 종료 프로젝트 목록 (취소선 + 불투명도 55%) 노출

### #1 Project Feed

**sm_notes 테이블** (신규 Supabase 마이그레이션 필요):
- `type`: `progress` / `issue` / `meeting` / `handoff`
- 미팅: `meeting_date`, `attendees[]`
- 인수인계: `to_user_id`, `task_id`

**FocusPanel 탭**:
- 프로젝트 선택 시 "일정 | 피드" 탭 전환 UI 표시
- 피드 탭: `ProjectFeed` 컴포넌트 (역시간순 노트 타임라인)

**NoteForm**:
- 타입 선택 (progress/issue/meeting/handoff) → 해당 필드 동적 노출
- meeting: 날짜 + 참석자 멀티선택
- handoff: 수신자 선택 + 연결 업무 선택
- issue: 연결 업무 선택

**ProjectFeed**:
- 노트 카드: 타입 아이콘 + 배지 + 작성자 + 시간 + 내용 + 연결 태스크 칩
- 인수인계: 발신자 → 수신자 화살표 표시
- 본인 작성 노트: 삭제 버튼 노출
