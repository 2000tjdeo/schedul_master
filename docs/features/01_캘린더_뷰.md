Schedule Master의 날짜 기반 시각화 컴포넌트 전체를 다룹니다.

## CalendarGrid.jsx — 월간 그리드

### 구조

```
CalendarGrid
├── 헤더 행 (일~토 요일 라벨)
└── 날짜 셀 (6주 × 7일 = 42칸)
    ├── TaskBar × MAX_BARS (기간 업무)
    ├── ItemChip × MAX_CHIPS (단일 업무 / 약속)
    └── "+N개 더" 오버플로 버튼
```

### 핵심 상수

```js
const MAX_BARS  = 3;  // 셀당 최대 TaskBar 표시 개수
const MAX_CHIPS = 3;  // 셀당 최대 ItemChip 표시 개수
```

### 날짜 범위 필터링

기간 업무(task_date ~ due_date)가 특정 날짜 `ymd`에 표시되어야 하는지 판별:

```js
// task_date <= ymd <= due_date 인 경우 해당 셀에 TaskBar 렌더링
const isInRange = task.task_date <= ymd && ymd <= task.due_date;
```

단일 업무 및 약속은 `task_date === ymd` 또는 `date === ymd` 로 정확히 일치하는 날에만 표시됩니다.

### 오버플로 처리

셀에 표시할 항목이 MAX 값을 초과하면 `"+N개 더"` 버튼을 렌더링합니다.

```jsx
{overflow > 0 && (
  <button onClick={() => openOverflowModal(date)}>
    +{overflow}개 더
  </button>
)}
```

## TaskBar.jsx — 기간 업무 바

- `task_date`부터 `due_date`까지 가로로 이어지는 색상 바
- 주(week) 경계에서 잘리고 다음 행에 이어짐
- 클릭 시 업무 상세 모달 오픈

```jsx
<TaskBar
  task={task}
  startCol={startDayOfWeek}   // 0(일)~6(토)
  span={spanDays}             // 해당 주에서 차지하는 일수
  color={task.color ?? '#4f46e5'}
/>
```

---

## ItemChip.jsx — 단일 업무 / 약속 칩

- 하루짜리 업무 또는 약속을 작은 칩 형태로 표시
- 우선순위 색상(low=초록, medium=노랑, high=빨강) 또는 약속 색상 적용
- 클릭 시 상세/편집 모달 오픈

---

## MiniCalendar.jsx — 모바일 미니 캘린더

### 다중 점(dot) 표시

날짜 아래 최대 3개의 색상 점으로 해당 날의 항목 유형을 표시합니다.

```js
// 점 배열 생성 규칙
const dots = [];
if (hasTasks)        dots.push({ color: '#4f46e5' }); // 업무: 인디고
if (hasAppointments) dots.push({ color: '#10b981' }); // 약속: 에메랄드
if (hasOverflow)     dots.push({ color: '#f59e0b' }); // 초과: 앰버
```

- 점이 3개를 초과하지 않도록 제한
- 선택된 날짜는 강조 스타일 적용

## WeekView.jsx — 주간 뷰

- 7개 열(일~토) × 24시간 행 구성
- `start_time` / `end_time` 기준으로 약속 블록 배치
- 시간 충돌 시 열 분할(side-by-side) 처리

---

## YearView.jsx — 연간 뷰

- 12개월 MiniCalendar 격자 표시
- 월별 업무/약속 밀도를 히트맵 색상으로 표현
- 월 클릭 시 해당 월의 CalendarGrid로 이동

---

## Scheduler.jsx — 시간표형 스케줄러

- 하루 단위 타임라인 뷰
- 시간 슬롯 클릭으로 빠른 약속 생성 지원
- `task_time` / `duration` 기반으로 블록 높이 계산

```js
// 블록 높이 = duration(분) / 60 * 셀높이(px)
const blockHeight = (duration / 60) * CELL_HEIGHT;
```

## 데이터 흐름 요약

```
Supabase
  └─ sm_tasks (task_date, due_date)
  └─ sm_appointments (date, start_time, end_time)
        │
        ▼
  store/taskStore.js (Zustand)
        │
        ▼
  CalendarGrid / WeekView / YearView / Scheduler
        │
        ├─ TaskBar    (기간: task_date ~ due_date)
        ├─ ItemChip   (단일: task_date = 해당일)
        └─ MiniCalendar (다중 점 표시)
```
