# 06. 관리자 칩 관리

> **관련 파일:** `client/src/pages/AdminPage.jsx`, Supabase `sm_chip_presets` 테이블

---

## 개요

Quick Menu에서 사용하는 제목 칩(title chip) 프리셋을 관리자가 직접 추가/삭제하는 페이지.
관리자(role='admin') 계정만 접근 가능.

---

## 접근 방법

TopBar → 설정(⚙) → Admin Dashboard

---

## sm_chip_presets 테이블

| 컬럼 | 타입 | 설명 |
|---|---|---|
| id | int | PK |
| label | text | 칩 표시 텍스트 |
| type | text | 현재 'title'만 사용 |
| created_at | timestamp | 생성일 |

---

## 주의사항

- **RLS(Row Level Security)** 비활성화 필요
  - Supabase 대시보드 → Authentication → Policies에서 `sm_chip_presets` RLS 비활성화
  - 활성화 상태면 읽기 실패 (데이터 빈 상태로 표시됨)
- 칩은 Quick Menu NLStrip의 `titlePresets`로 표시됨
- DB 데이터 없으면 폴백(fallback) 텍스트로 한국어 기본값 표시

---
