# 06. 관리자 칩 관리

Quick Menu에서 사용할 제목 칩 프리셋을 관리하는 관리자 페이지입니다.

---

## 관련 파일

| 파일 | 경로 | 역할 |
|------|------|------|
| `AdminPage` | `src/pages/AdminPage.jsx` | 칩 관리 UI |
| Supabase 테이블 | `sm_chip_presets` | 칩 데이터 저장 |

---

## 기능 개요

Quick Menu의 **제목칩(title chip)** 프리셋을 관리자가 직접 추가·수정·삭제하는 페이지입니다.
현재 `type = 'title'` 인 칩만 Quick Menu에 표시됩니다.

---

## sm_chip_presets 테이블

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `id` | uuid | PK |
| `label` | text | 칩 표시 텍스트 (사용자에게 보이는 이름) |
| `type` | text | `'title'` (현재는 title 타입만 Quick Menu에서 사용) |
| `value` | text | 실제 폼에 적용되는 값 |
| `created_at` | timestamp | 생성 일시 |

---

## CRUD 동작

### 읽기 (Read)

```js
const { data, error } = await supabase
  .from('sm_chip_presets')
  .select('*')
  .eq('type', 'title')
  .order('created_at', { ascending: true });
```

### 추가 (Create)

```js
await supabase
  .from('sm_chip_presets')
  .insert({ label: newLabel, type: 'title', value: newLabel });
```

### 삭제 (Delete)

```js
await supabase
  .from('sm_chip_presets')
  .delete()
  .eq('id', chipId);
```

## Quick Menu 연동

AdminPage에서 저장한 칩 데이터는 Quick Menu의 제목칩 팔레트에 바로 반영됩니다.

```js
// UnifiedCreateModal.jsx 또는 NLStrip.jsx
const [titlePresets, setTitlePresets] = useState([]);

useEffect(() => {
  supabase
    .from('sm_chip_presets')
    .select('*')
    .eq('type', 'title')
    .then(({ data }) => setTitlePresets(data ?? []));
}, []);

// Quick Menu 렌더링
{titlePresets.map(chip => (
  <button key={chip.id} onClick={() => applyChip(chip.value)}>
    {chip.label}
  </button>
))}
```

## Supabase RLS 설정 주의사항

`sm_chip_presets` 테이블에 **Row Level Security(RLS)가 활성화**되어 있으면
인증되지 않은 사용자는 데이터를 읽지 못해 칩이 빈 상태로 표시됩니다.

### 해결 방법

1. Supabase 대시보드 접속
2. `Table Editor` → `sm_chip_presets` 선택
3. `Authentication` → `Policies` 탭
4. RLS 비활성화 또는 `SELECT` 허용 정책 추가

```sql
-- 모든 사용자가 칩 프리셋을 읽을 수 있도록 허용
CREATE POLICY "allow_read_chip_presets"
  ON sm_chip_presets
  FOR SELECT
  USING (true);
```

## 폴백 기본값 (DB 데이터 없을 때)

DB에 데이터가 없거나 RLS로 읽기가 실패한 경우, 하드코딩된 기본값을 사용합니다.

| 탭 | 기본 칩 목록 |
|----|------------|
| 업무 탭 | 기획 회의, 견적 제출, 납품 확인, API 개발 |
| 약속 탭 | 클라이언트 미팅, 팀 회의, 현장 방문, 디자인 검토 |

## type 필드 확장 계획

현재는 `type = 'title'` 만 사용하지만, 향후 아래 타입으로 확장 예정입니다.

| type | 용도 |
|------|------|
| `'title'` | 제목 칩 (현재 구현) |
| `'category'` | 카테고리 칩 |
| `'tag'` | 태그 칩 |
