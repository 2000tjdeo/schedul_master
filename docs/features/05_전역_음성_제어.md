# 05. 전역 음성 제어

Web Speech API를 활용한 음성 입력 및 전역 음성 명령 시스템입니다.

---

## 현재 구현 상태 vs 설계

| 구분 | 상태 | 설명 |
|------|------|------|
| `useSpeech.js` | 구현 완료 | NLStrip 내 음성 입력 |
| `useGlobalVoice.js` | 설계 중 | 앱 레벨 전역 음성 명령 |
| TopBar 글로벌 마이크 버튼 | 설계 중 | 어디서든 음성 명령 트리거 |
| `VoiceCommandOverlay` | 설계 중 | 인식 중 풀스크린 오버레이 |

## hooks/useSpeech.js — 현재 구현

Web Speech API의 `SpeechRecognition`을 React 훅으로 래핑합니다.

### 사용 언어

```js
recognition.lang = 'ko-KR'; // 한국어 인식
```

### 훅 인터페이스

```js
const {
  isListening,     // boolean: 현재 인식 중 여부
  transcript,      // string: 인식된 텍스트
  startListening,  // () => void: 인식 시작
  stopListening,   // () => void: 인식 중단
  error,           // string | null: 오류 메시지
} = useSpeech();
```

### 구현 예시

```js
// hooks/useSpeech.js
import { useState, useEffect, useRef } from 'react';

export function useSpeech() {
  const [isListening, setIsListening] = useState(false);
  const [transcript, setTranscript]   = useState('');
  const [error, setError]             = useState(null);
  const recognitionRef = useRef(null);

  useEffect(() => {
    const SpeechRecognition =
      window.SpeechRecognition || window.webkitSpeechRecognition;

    if (!SpeechRecognition) {
      setError('이 브라우저는 음성 인식을 지원하지 않습니다.');
      return;
    }

    const recognition = new SpeechRecognition();
    recognition.lang = 'ko-KR';
    recognition.continuous = false;
    recognition.interimResults = false;

    recognition.onresult = (e) => {
      setTranscript(e.results[0][0].transcript);
    };
    recognition.onend  = () => setIsListening(false);
    recognition.onerror = (e) => setError(e.error);

    recognitionRef.current = recognition;
  }, []);

  const startListening = () => {
    setTranscript('');
    setError(null);
    recognitionRef.current?.start();
    setIsListening(true);
  };

  const stopListening = () => {
    recognitionRef.current?.stop();
  };

  return { isListening, transcript, startListening, stopListening, error };
}
```

### 현재 사용 위치

```
NLStrip.jsx
  └─ useSpeech()
      └─ 마이크 버튼 클릭 → startListening()
          └─ transcript 변경 → parseNLWithGemini(transcript)
```

## hooks/useGlobalVoice.js — 설계 (미구현)

앱 어디서든 호출 가능한 전역 음성 명령 훅입니다.

### 지원 명령어

| 음성 명령 | 동작 |
|----------|------|
| `"새 업무"` | UnifiedCreateModal 열기 (업무 탭) |
| `"새 약속"` | UnifiedCreateModal 열기 (약속 탭) |
| `"캘린더"` | 캘린더 뷰로 이동 |
| `"보드"` | 칸반보드로 이동 |
| `"오늘"` | 오늘 날짜로 포커스 이동 |
| 그 외 텍스트 | NLStrip에 텍스트 전달 후 NLP 파싱 |

### 설계된 인터페이스

```js
// hooks/useGlobalVoice.js (설계)
export function useGlobalVoice({ onNavigate, onOpenModal, onNLText }) {
  const { transcript, startListening, isListening } = useSpeech();

  useEffect(() => {
    if (!transcript) return;

    // 명령어 라우팅
    if (transcript.includes('새 업무'))   return onOpenModal('task');
    if (transcript.includes('새 약속'))   return onOpenModal('appointment');
    if (transcript.includes('캘린더'))    return onNavigate('calendar');
    if (transcript.includes('보드'))      return onNavigate('kanban');
    if (transcript.includes('오늘'))      return onNavigate('today');

    // 명령어 아닌 경우 NL 텍스트로 처리
    onNLText(transcript);
  }, [transcript]);

  return { startListening, isListening };
}
```

## TopBar 글로벌 마이크 버튼 — 설계 (미구현)

TopBar에 항상 노출되는 마이크 버튼으로 전역 음성 제어를 시작합니다.

```jsx
// src/components/TopBar.jsx (설계)
import { useGlobalVoice } from '../hooks/useGlobalVoice';

function TopBar() {
  const { startListening, isListening } = useGlobalVoice({
    onNavigate:  (view) => setCurrentView(view),
    onOpenModal: (type) => openCreateModal(type),
    onNLText:    (text) => setGlobalNLText(text),
  });

  return (
    <header>
      {/* ... 기존 TopBar 요소 ... */}
      <button
        onClick={startListening}
        className={isListening ? 'animate-pulse text-red-500' : ''}
        title="음성 명령 (ko-KR)"
      >
        🎤
      </button>
    </header>
  );
}
```

## VoiceCommandOverlay — 설계 (미구현)

음성 인식 중 사용자에게 시각적 피드백을 제공하는 풀스크린 오버레이입니다.

```jsx
// src/components/VoiceCommandOverlay.jsx (설계)
function VoiceCommandOverlay({ isListening, transcript }) {
  if (!isListening) return null;

  return (
    <div className="fixed inset-0 bg-black/60 flex flex-col items-center justify-center z-50">
      {/* 애니메이션 마이크 아이콘 */}
      <div className="w-24 h-24 rounded-full bg-red-500 animate-pulse flex items-center justify-center">
        🎤
      </div>

      {/* 인식 중 안내 */}
      <p className="text-white text-xl mt-6">말씀하세요...</p>

      {/* 실시간 인식 텍스트 */}
      {transcript && (
        <p className="text-white/80 mt-4 text-lg">{transcript}</p>
      )}

      {/* 지원 명령어 힌트 */}
      <div className="mt-8 text-white/60 text-sm space-y-1 text-center">
        <p>"새 업무" · "새 약속" · "캘린더" · "보드" · "오늘"</p>
        <p>또는 자연어로 일정을 말씀하세요</p>
      </div>
    </div>
  );
}
```

## 전체 아키텍처 (설계 완성 후)

```
TopBar 마이크 버튼
        │ click
        ▼
useGlobalVoice (App 레벨)
        │
        ├─ useSpeech (ko-KR 인식)
        │       │ transcript
        │       ▼
        │  명령어 파싱
        │       ├─ "새 업무" → onOpenModal('task')
        │       ├─ "새 약속" → onOpenModal('appointment')
        │       ├─ "캘린더" → onNavigate('calendar')
        │       ├─ "보드"   → onNavigate('kanban')
        │       ├─ "오늘"   → onNavigate('today')
        │       └─ 기타     → onNLText → NLStrip → Gemini
        │
        └─ VoiceCommandOverlay (인식 중 풀스크린 표시)
```
