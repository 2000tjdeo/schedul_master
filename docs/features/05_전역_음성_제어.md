# 05. 전역 음성 제어

> **관련 파일:** `client/src/hooks/useGlobalVoice.js`, `client/src/hooks/useSpeech.js`, `client/src/components/TopBar/TopBar.jsx`, `client/src/App.jsx`

---

## 개요

앱 전체에서 마이크 버튼 하나로 음성 명령을 처리할 수 있는 기능.
기존에는 업무/약속 생성 모달(NLStrip) 안에서만 음성 입력이 가능했으나,
TopBar의 글로벌 마이크 버튼으로 어디서든 명령 가능.

---

## 파일 구조

```
hooks/
  useSpeech.js        ← Web Speech API 기본 래퍼 (ko-KR)
  useGlobalVoice.js   ← 음성→명령 파싱 & 라우팅 (신규)

components/TopBar/
  TopBar.jsx          ← 마이크 버튼 UI (voiceStatus props)

App.jsx               ← useGlobalVoice 연결, 오버레이 렌더링
```

---

## 지원 명령어

| 말하는 내용 | 동작 |
|---|---|
| "새 업무" / "업무 추가" / "업무 만들어" | 업무 생성 모달 열기 |
| "새 약속" / "약속 추가" / "약속 만들어" | 약속 생성 모달 열기 |
| "캘린더" / "달력" | 캘린더 탭으로 이동 |
| "보드" / "칸반" | 칸반 보드로 이동 |
| "오늘" | 오늘 날짜 선택 |
| "내일 오후 2시 팀 미팅" (자연어) | parseNL 파싱 → 업무 모달 자동 열기 |

---

## 상태 흐름

```
idle → [마이크 버튼 클릭] → listening → [음성 입력] → processing → [명령 실행] → idle
                                          ↑
                                    오버레이 표시 (배경 블러 + 힌트 카드)
```

---

## 마이크 버튼 상태별 색상

| 상태 | 아이콘 | 색상 |
|---|---|---|
| idle | `mic_none` | 회색 (#71717a) |
| listening | `mic` (pulse 애니) | 빨강 (#ef4444) |
| processing | `hourglass_top` | 노랑 (#f59e0b) |

---

## 주요 코드 위치

- `useGlobalVoice.js:handleResult()` — 명령어 키워드 매칭 & 라우팅
- `App.jsx:handleVoiceToggle()` — 시작/중지 토글
- `App.jsx` 오버레이 렌더 — `voiceStatus === 'listening'` 시 풀스크린
